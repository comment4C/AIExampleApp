### Project setup

Do you have a virtual device for run this program

#### Install
##### Android
```
npm install -g react-native-cli
```
##### iOS
```
brew install node
brew install watchman
```

#### Run
##### Android
```
npx react-native run-android
```
##### iOS
```
npx react-native run-ios
```
